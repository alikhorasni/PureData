# PureData
Ultra‑fast, adaptive data cleansing &amp; quality engine. Handles 100+ GB datasets with zero‑copy, lazy execution. Auto‑detects distribution drift, missing data, outliers, and applies optimal fixes. Built on Polars &amp; Apache Arrow for unmatched speed. 1‑line magic: `puredata clean dirty.csv clean.parquet --auto-magic`. MIT‑licensed.
